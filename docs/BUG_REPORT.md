# libbsem_math.dna Bug Report - Full Scan (2053 lines)

## CRITICAL (输出错误)

### Bug 1: 矩阵设置位置错误 (Line 141-143)
```
NUM st[50] 1   # M00=1  ✓
NUM st[55] 2   # M12=2  ✗ 应该是 M01=st[51]=0
NUM st[58] 3   # M22=3  ✗ 应该是 M02=st[52]=0
```
**影响**: 单位矩阵 I*[1,2,3] 输出 [0,0,0] 而不是 [1,2,3]
**修复**: 正确设置 3x3 单位矩阵 + 输入向量

### Bug 2: 矩阵非对角线元素未清零 (Line 141-150)
主程序只设置了 st[50], st[55], st[58] 和 st[59-61]，其余矩阵元素残留之前测试的值。
**影响**: M01(st[51]), M02(st[52]), M10(st[53]) 等含垃圾值
**修复**: 完整设置所有9个矩阵元素

## MEDIUM (权重设计不完美)

### Bug 3: XOR 网络输出范围不够分明
1000-scale 下权重过大，ReLU 将 ±1000 压缩为 0/1000，导致输出只有 425(同) 和 650(不同)。
**影响**: XOR 无法清晰区分 0/1
**修复**: 缩小权重到 ±100 范围

## LOW ( tube 污染)

### Bug 4: st[51], st[52] 残留值
vec_dot 测试后 st[51]=2, st[52]=3 残留到 mat_vec 测试。
**影响**: 如果 Bug 2 修复，这个不再影响（会显式清零）
