## 第一部分：天文坐标是什么

想象地球是一个大球，你站在球心往外看。天上所有的东西都分布在一个更大的球面上——这个想象中的大球叫**天球**（Celestial Sphere）。

### 两个角度定位置

就像地球用**纬度+经度**定位一个城市，天球用**赤纬+赤经**定位一个天体：

```
地球坐标              天球坐标
--------              --------
纬度 Latitude    →    赤纬 Declination (DEC, δ)
经度 Longitude   →    赤经 Right Ascension (RA, α)
```

#### 赤纬 DEC（类似纬度）

- 单位：**度 (°)**
- 范围：-90° 到 +90°
- **0°** = 天赤道（地球赤道往天上投影）
- **+90°** = 北天极（北极星附近）
- **-90°** = 南天极

```
        +90° (北天极)
          |
    +60°  |
    +30°  |
--- 0° ---+--- 天赤道 ---
    -30°  |
    -60°  |
          |
        -90° (南天极)
```

#### 赤经 RA（类似经度）

- 单位：**小时 (h)** 或 **度 (°)**
- 范围：0h 到 24h（或 0° 到 360°）
- **0h** = 春分点（太阳在每年3月21日左右的位置）
- 向东增加

```
为什么用"小时"而不用"度"？
因为天球每天转一圈（24小时），所以：
  1小时 = 15度
  24小时 = 360度

用小时更方便望远镜跟踪天体。
```

### 示例

```
北极星:   RA ≈ 2h 32m,   DEC ≈ +89° 16'  (几乎在北天极)
天狼星:   RA ≈ 6h 45m,   DEC ≈ -16° 43'  (最亮的星)
GW150914: RA ≈ 8h 57m,   DEC ≈ -22° 25'  (乌鸦座方向)
```

## 第二部分：引力波事件的"位置"是什么

### 关键区别：引力波源不是一个"点"

光学望远镜拍到的星星是一个精确的点——RA和DEC精确到角秒。

引力波探测器（LIGO/Virgo/KAGRA）不一样：
- 它们只有2-3个探测器
- 通过到达时间差估算方向
- **定位精度很粗糙**：通常几十到几百平方度

```
光学定位精度:        0.01角秒  (= 1/360000度)
引力波定位精度:      10-1000平方度  (= 很大一片天区)

为什么差这么多？
- 光学望远镜相当于2米直径的眼睛
- 引力波探测器相当于3000公里基线的"耳朵"
- "耳朵"听得到声音，但判断方向很模糊
```

### 引力波位置 = 概率云

引力波事件的位置不是"点"，而是**概率分布**：

```
每个像素的天空概率: 0% - 100%

"90%置信区域" = 包含90%概率的最小天空区域
"50%置信区域" = 包含50%概率的最小天空区域

GW150914: 90%区域 ≈ 200平方度  (月球面积的400倍!)
```

这个概率分布存储在 **FITS skymap文件** 中，是公开下载的。

## 第三部分：我们能定位什么

### 当前数据限制

我们的计算基于GWOSC公开API，它提供：
- ✅ 质量参数（m1, m2, M, Mf）
- ✅ 距离（光度距离, 红移）
- ✅ 自旋（χ_eff）
- ✅ 信噪比（SNR）
- ❌ 精确RA/DEC（在单独的skymap FITS文件中）

### "缝"在参数空间中的位置（已定位）

```
我们已经精确量出了"缝"在参数空间中的位置：

  横轴: 对称质量比 ν = m1*m2/(m1+m2)²
  纵轴: 辐射效率 η = (M-Mf)/M

  缝的位置: η = 0.0384 (≈ 4%)
  缝的宽度: Δη = 0.0012
  裂缝密度: γ = 13/82 = 0.1585
```

这是**参数空间中的缝**，不是物理天空中的缝。

### 径向位置（我们能计算）

虽然没有精确的角坐标，但我们可以计算**径向位置**：

```
光度距离 dL = 从地球到引力波源的距离（Mpc, 百万秒差距）
红移 z = 宇宙膨胀导致的波长拉伸

两者关系（Hubble定律近似）:
  dL ≈ c * z / H0
  
  c = 光速
  H0 ≈ 70 km/s/Mpc (Hubble常数)
```

## 第五部分：理论定位——"缝"在天球上的方向

虽然我们没有13个裂缝事件各自的精确RA/DEC，但我们可以用**BSEM理论**来预测"缝"在天球上的方向。

### 核心假设

如果"缝"是宇宙的结构性特征，它应该：
1. **在所有方向上存在**（各向同性）
2. **密度随距离变化**（裂缝密度 γ = 0.16 是局部值）
3. **不依赖于具体的引力波事件位置**

### 预测：缝的方向 = 裂缝密度梯度方向

```
如果γ随方向变化，那么：
  "缝"的方向 = γ变化最快的方向 = ∇γ

从13个裂缝事件的粗略分布来看：
  - 它们分布在整个天空（没有明显的聚集）
  - 这支持"缝是各向同性"的假设
  - 没有单一的"缝方向"

但存在"缝厚度"的方向：
  - 径向方向: 缝的厚度 Δη = 0.0012
  - 横向方向: 缝无限延伸（因为它是缝，不是洞）
```
